# 📖 Vegan Scanner <a name="about-project"></a>

**Vegan Scanner** is a mobile web app that allows users to scan and search for products suitable for vegans. Users can scan bar or QR codes to find out what products are suitable for vegans, also they can check their nutritional quality level.

## 🛠 Built With <a name="built-with"></a>

1. React
2. Jest
3. React-Redux

### Tech Stack <a name="tech-stack"></a>

In this project I used React to build a Vegan Scanner web app.

  </ul>
</details>

### Key Features <a name="key-features"></a>

- Scan bar and QR codes.
- Filter out products by name or brand.
- Show detailed nutritional information for each product.

# Getting Started <a name="getting-started"></a>

In this project it is used React to build a Vegan Scanner mobile web app.

### Setup

To get a local copy, follow these steps.

Clone this repository to your desired folder:

  ```sh
  cd my-folder
  git clone https://github.com/ivangonzalez224/Vegan-scanner.git
  code . (To open in Vscode)
```

### Prerequisites

In order to run this project you need:

1. Git.
2. Visual Studio Code or another code editor.
3. Browser.  
  
  
### Install

This project does not require installation. Just clone the repository to your local machine.
Even though, for general purposes, Install this project with:
```sh
  cd my-folder
  npm install
```

### Usage

To run the project, do the following:
- Run `npm start` to open the project in your browser.
